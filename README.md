# Instagram Media Downloader Bot

Telegram бот на базе `aiogram 3.x` для безопасного скачивания контента из Instagram (Reels, Posts, Stories).

## ✨ Возможности

- 📹 **Скачивание Reels**: В оригинальном качестве без водяных знаков.
- 🖼️ **Скачивание постов**: Поддержка одиночных фото и видео.
- 📚 **Поддержка каруселей**: Автоматическая разбивка постов (10+ элементов) на несколько альбомов.
- 🔒 **Безопасность**: Работа через сторонний RapidAPI (нет риска блокировки личного Instagram-аккаунта).
- 💰 **Монетизация**: Система лимитов (3 бесплатных скачивания) и Premium доступ.
- 🛡️ **Админ-панель**: Безлимитный доступ для администратора.

## 🛠 Установка

1. **Клонируйте проект** или скопируйте файлы в папку.
2. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Настройте переменные окружения**:
   Создайте файл `.env` (или используйте существующий) и заполните следующие поля:
   - `BOT_TOKEN`: Токен вашего бота от @BotFather.
   - `RAPIDAPI_KEY`: Ваш API ключ с платформы RapidAPI.
   - `RAPIDAPI_HOST`: Хост API (например, `instagram-scraper-20251.p.rapidapi.com`).
   - `ALLOWED_USERS`: ID администраторов через запятую (для безлимита).

## 🚀 Запуск

```bash
python bot.py
```

## 🗄 База данных

Проект использует SQLite (`bot_data.db`) для хранения:
- Количества успешных запросов пользователей.
- Премиум-статуса.

## ⚠️ Важные примечания

- Бот **не поддерживает** скачивание из приватных (закрытых) аккаунтов.
- Лимит Telegram на один альбом — 10 файлов (бот автоматически обходит это, разделяя сообщения).
